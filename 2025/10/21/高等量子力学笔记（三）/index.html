高等量子力学笔记（三）：量子化条件

量子泊松括号

经典类比

由于量子力学是比经典力学更为基础的理论，量子力学中的规律应在经典力学中有对应的经典简化表现。因此我们可以通过经典力学中的一些概念来类比理解量子力学中的对应概念。经典力学中，系统的状态由相空间中的点 (q,p)(q,p)(q,p) 描述，物理量由相空间中的函数 μ(q,p)\mu(q,p)μ(q,p) 描述。两个物理量 μ\muμ 和 ν\nuν 的泊松括号定义为

[μ,ν]PB=∑i=1N(∂μ∂qi∂ν∂pi−∂μ∂pi∂ν∂qi)\left[\mu, \nu\right]_{\text{PB}} =\sum_{i=1}^{N}\left(\frac{\partial \mu}{\partial q_i} \frac{\partial \nu}{\partial p_i}-\frac{\partial \mu}{\partial p_i} \frac{\partial \nu}{\partial q_i}\right) [μ,ν]PB​=i=1∑N​(∂qi​∂μ​∂pi​∂ν​−∂pi​∂μ​∂qi​∂ν​)

其中 NNN 是系统的自由度数目。泊松括号满足以下性质：

  1. 反对称性：[μ,ν]PB=−[ν,μ]PB\left[\mu, \nu\right]_{\text{PB}} = -\left[\nu, \mu\right]_{\text{PB}}[μ,ν]PB​=−[ν,μ]PB​
  2. 线性：[aμ+bν,λ]PB=a[μ,λ]PB+b[ν,λ]PB\left[a\mu + b\nu, \lambda \right]_{\text{PB}} = a\left[\mu, \lambda\right]_{\text{PB}} + b\left[\nu, \lambda\right]_{\text{PB}}[aμ+bν,λ]PB​=a[μ,λ]PB​+b[ν,λ]PB​
  3. 乘积法则：[μν,λ]PB=μ[ν,λ]PB+ν[μ,λ]PB\left[\mu\nu, \lambda\right]_{\text{PB}} = \mu\left[\nu, \lambda\right]_{\text{PB}} + \nu\left[\mu, \lambda\right]_{\text{PB}}[μν,λ]PB​=μ[ν,λ]PB​+ν[μ,λ]PB​
  4. 雅可比恒等式：[μ,[ν,λ]PB]PB+[ν,[λ,μ]PB]PB+[λ,[μ,ν]PB]PB=0\left[\mu, \left[\nu, \lambda\right]_{\text{PB}}\right]_{\text{PB}} + \left[\nu, \left[\lambda, \mu\right]_{\text{PB}}\right]_{\text{PB}} + \left[\lambda, \left[\mu, \nu\right]_{\text{PB}}\right]_{\text{PB}} = 0[μ,[ν,λ]PB​]PB​+[ν,[λ,μ]PB​]PB​+[λ,[μ,ν]PB​]PB​=0

量子泊松括号

但在量子力学中，μ\muμ 和 ν\nuν 的乘法不满足交换律，因此无法直接使用经典力学中的泊松括号定义。接下来我们将通过上述条件来唯一地确定量子泊松括号。根据经典泊松括号的乘法法则，我们对比以下两个量子泊松括号：

[μ1μ2,ν1ν2]=μ1[μ2,ν1ν2]+[μ1,ν1ν2]μ2=μ1ν1[μ2,ν2]+μ1[μ2,ν1]ν2+ν1[μ1,ν2]μ2+[μ1,ν1]ν2μ2\left[\mu_1\mu_2, \nu_1\nu_2\right] = \mu_1\left[\mu_2, \nu_1\nu_2\right] + \left[\mu_1, \nu_1\nu_2\right]\mu_2= \mu_1\nu_1\left[\mu_2, \nu_2\right] + \mu_1\left[\mu_2, \nu_1\right]\nu_2 + \nu_1\left[\mu_1, \nu_2\right]\mu_2 + \left[\mu_1, \nu_1\right]\nu_2\mu_2 [μ1​μ2​,ν1​ν2​]=μ1​[μ2​,ν1​ν2​]+[μ1​,ν1​ν2​]μ2​=μ1​ν1​[μ2​,ν2​]+μ1​[μ2​,ν1​]ν2​+ν1​[μ1​,ν2​]μ2​+[μ1​,ν1​]ν2​μ2​

[μ1μ2,ν1ν2]=[μ1,ν1ν2]μ2+μ1[μ2,ν1ν2]=ν1[μ1,ν2]μ2+[μ1,ν1]ν2μ2+μ1ν1[μ2,ν2]+μ1[μ2,ν1]ν2\left[\mu_1\mu_2, \nu_1\nu_2\right] = \left[\mu_1, \nu_1\nu_2\right]\mu_2 + \mu_1\left[\mu_2, \nu_1\nu_2\right]= \nu_1\left[\mu_1, \nu_2\right]\mu_2 + \left[\mu_1, \nu_1\right]\nu_2\mu_2 + \mu_1\nu_1\left[\mu_2, \nu_2\right] + \mu_1\left[\mu_2, \nu_1\right]\nu_2 [μ1​μ2​,ν1​ν2​]=[μ1​,ν1​ν2​]μ2​+μ1​[μ2​,ν1​ν2​]=ν1​[μ1​,ν2​]μ2​+[μ1​,ν1​]ν2​μ2​+μ1​ν1​[μ2​,ν2​]+μ1​[μ2​,ν1​]ν2​

两式右边应是相等的，我们有

μiνi−νiμi=c[μi,νi]Q\mu_i\nu_i-\nu_i\mu_i= c \left[\mu_i, \nu_i\right]_{\text{Q}} μi​νi​−νi​μi​=c[μi​,νi​]Q​

经典力学：两个实变量组成的 [μi,νi]PB\left[\mu_i, \nu_i\right]_{\text{PB}}[μi​,νi​]PB​ 应是实数

量子力学：两个厄米算符组成的 [μi,νi]Q\left[\mu_i, \nu_i\right]_{\text{Q}}[μi​,νi​]Q​ 也应是厄米算符

其中 ccc 必须是一个数。又因为 μiνi−νiμi\mu_i\nu_i-\nu_i\mu_iμi​νi​−νi​μi​ 是反厄米的，而 [μi,νi]Q\left[\mu_i, \nu_i\right]_{\text{Q}}[μi​,νi​]Q​ 是厄米的，因此 ccc 必须是一个纯虚数。我们取 c=iℏc=i\hbarc=iℏ，从而得到量子泊松括号的定义：

[μ,ν]Q=1iℏ(μν−νμ)=1iℏ[μ,ν]\left[\mu, \nu\right]_{\text{Q}} = \frac{1}{i\hbar} \left(\mu\nu - \nu\mu\right)= \frac{1}{i\hbar} [\mu, \nu] [μ,ν]Q​=iℏ1​(μν−νμ)=iℏ1​[μ,ν]

其中 ℏ\hbarℏ 具有作用量的量纲：[ℏ]=[q][p]=[E][t][\hbar] = [q][p]=[E][t][ℏ]=[q][p]=[E][t]。为使理论与实验相符，我们必须取 ℏ=h/2π\hbar=h/2\piℏ=h/2π。

量子条件

设想：量子泊松括号与经典泊松括号具有相同的值：[μ,ν]Q=[μ,ν]PB[\mu, \nu]_{\text{Q}} = [\mu, \nu]_{\text{PB}}[μ,ν]Q​=[μ,ν]PB​。这意味着

[μ,ν]PB⟷[μ,ν]Q=1iℏ[μ,ν][\mu, \nu]_{\text{PB}} \longleftrightarrow [\mu,\nu]_\text{Q} = \frac{1}{i\hbar} [\mu, \nu] [μ,ν]PB​⟷[μ,ν]Q​=iℏ1​[μ,ν]

那么量子泊松括号也满足

[q^i,q^j]=[p^i,p^j]=0[\hat{q}_i, \hat{q}_j]=[\hat{p}_i,\hat{p}_j] = 0 [q^​i​,q^​j​]=[p^​i​,p^​j​]=0

[q^i,p^j]=iℏδij[\hat{q}_i, \hat{p}_j] = i\hbar \delta_{ij} [q^​i​,p^​j​]=iℏδij​

这就是基本量子条件（正则对易关系）。

  • 基本量子条件只能应用于有经典对应的动力学系统（能用 qqq 和 ppp 描述）。但是量子力学中的自旋等内禀自由度没有经典对应，因此不能用基本量子条件来描述。
  • 基本量子条件为量子力学和经典力学之间建立了桥梁，经典力学可以看作量子力学在 ℏ→0\hbar \to 0ℏ→0 下的极限情况。
  • 量子泊松括号是一个纯粹的代数概念，因此是比经典泊松括号更为基本的概念，后者只能通过参考 𝑝 和 𝑞 来定义。由于这个原因，在量子力学中，正则坐标和动量的核心地位被削弱。
  • [q^i,p^j]=iℏδij[\hat{q}_i, \hat{p}_j] = i\hbar \delta_{ij}[q^​i​,p^​j​]=iℏδij​ 表明不同自由度的动力学变量是对易的。凭借物理直觉，我们或许能够找到某些动力学系统的量子条件，即使这些系统中并不存在 qqq 和 ppp。
