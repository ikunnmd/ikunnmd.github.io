密度矩阵的定义和性质

本征态下的密度矩阵

对于一个处于热平衡态的态矢 ∣E⟩\ket{E}∣E⟩，它的概率为

P(E)T=e−βEZP(E)_T = \frac{e^{-\beta E}}{Z} P(E)T​=Ze−βE​

其中 Z=∑Ee−βEZ = \sum_E e^{-\beta E}Z=∑E​e−βE 是配分函数，起到归一化的作用，使得 ∑EP(E)T=1\sum_E P(E)_T = 1∑E​P(E)T​=1。
现考虑一个可观测量 QQQ，它的期望值为

⟨Q⟩T=∑EP(E)T⏟热概率⟨E∣Q^∣E⟩⏟量子概率\langle Q \rangle_T = \sum_E \underbrace{P(E)_T}_{\text{热概率}} \underbrace{\langle E | \hat{Q} | E \rangle}_{\text{量子概率}} ⟨Q⟩T​=E∑​热概率P(E)T​​​量子概率⟨E∣Q^​∣E⟩​​

上式的意义是在抽中态 ∣E⟩\ket{E}∣E⟩ 的情况下，计算 QQQ 的期望值。对其进行具体计算，则 ⟨Q⟩\langle Q \rangle⟨Q⟩ 可写为

⟨Q⟩=∑Ee−βEZ⟨E∣Q^∣E⟩=1Z∑E⟨E∣e−βH^Q^∣E⟩=1ZTr(Q^e−βH^)\langle Q \rangle = \sum_E \frac{e^{-\beta E}}{Z} \langle E | \hat{Q} | E \rangle = \frac{1}{Z} \sum_E \langle E | e^{-\beta \hat{H}} \hat{Q} | E \rangle = \frac{1}{Z} \text{Tr}(\hat{Q}e^{-\beta \hat{H}} ) ⟨Q⟩=E∑​Ze−βE​⟨E∣Q^​∣E⟩=Z1​E∑​⟨E∣e−βH^Q^​∣E⟩=Z1​Tr(Q^​e−βH^)

再从另一个角度计算 QQQ 的期望值

⟨Q⟩T=∑E∑Q′P(E)T⟨E∣Q∣Q′⟩⟨Q′∣E⟩=∑E∑Q′P(E)TQ′⟨Q′∣E⟩⟨E∣Q′⟩=∑Q′Q′⟨Q′∣(∑EP(E)T∣E⟩⟨E∣)∣Q′⟩=∑Q′Q′⟨Q′∣ρ^T∣Q′⟩\begin{align*} \langle Q \rangle_T &= \sum_E \sum_{Q'} P(E)_T \bra{E}Q \ket{Q'} \braket{Q'|E} = \sum_E \sum_{Q'} P(E)_T Q' \braket{Q'|E} \braket{E|Q'}\\ &=\sum_{Q'} Q' \bra{Q'} \left( \sum_E P(E)_T \ket{E}\bra{E} \right) \ket{Q'}=\sum_{Q'} Q' \bra{Q'}\hat{\rho}_T\ket{Q'} \end{align*} ⟨Q⟩T​​=E∑​Q′∑​P(E)T​⟨E∣Q∣Q′⟩⟨Q′∣E⟩=E∑​Q′∑​P(E)T​Q′⟨Q′∣E⟩⟨E∣Q′⟩=Q′∑​Q′⟨Q′∣(E∑​P(E)T​∣E⟩⟨E∣)∣Q′⟩=Q′∑​Q′⟨Q′∣ρ^​T​∣Q′⟩​

其中 ρ^T=∑EP(E)T∣E⟩⟨E∣=∑Ee−βEZ∣E⟩⟨E∣=e−βH^/Z\hat{\rho}_T = \sum_E P(E)_T \ket{E}\bra{E}= \sum_E \frac{e^{-\beta E}}{Z} \ket{E}\bra{E}=e^{-\beta \hat{H}}/Zρ^​T​=∑E​P(E)T​∣E⟩⟨E∣=∑E​Ze−βE​∣E⟩⟨E∣=e−βH^/Z，称为热系综的密度矩阵，也就是权重为玻尔兹曼分布的投影算符的加和。因此我们得到一个十分重要的结论：

⟨Q⟩T=Tr(Q^ρ^T)=Tr(ρ^TQ^)\langle Q \rangle_T = \text{Tr}(\hat{Q} \hat{\rho}_T )= \text{Tr}(\hat{\rho}_T \hat{Q}) ⟨Q⟩T​=Tr(Q^​ρ^​T​)=Tr(ρ^​T​Q^​)

密度矩阵 ρ^T=∑EP(E)T∣E⟩⟨E∣\hat{\rho}_T=\sum_E P(E)_T \ket{E}\bra{E}ρ^​T​=∑E​P(E)T​∣E⟩⟨E∣ 具有以下几个重要性质：

密度矩阵的一般定义

更一般的定义：若有一套正交完备基 {∣m⟩}\{\ket{m}\}{∣m⟩}，系统处于每个纯态的概率为 PmP_mPm​，满足 0≤Pm≤10 \leq P_m \leq 10≤Pm​≤1 且 ∑mPm=1\sum_m P_m = 1∑m​Pm​=1，则定义密度矩阵为

ρ=∑mPm∣m⟩⟨m∣\rho=\sum_m P_m \ket{m}\bra{m} ρ=m∑​Pm​∣m⟩⟨m∣

那么有

  1. ∣m⟩\ket{m}∣m⟩ 是本征值为 PmP_mPm​ 的本征态，且 0≤Pm≤10 \leq P_m \leq 10≤Pm​≤1，

ρ^∣m⟩=Pm∣m⟩\hat{\rho}\ket{m} = P_m \ket{m} ρ^​∣m⟩=Pm​∣m⟩

  1. Tr(ρ)=1\text{Tr}(\rho) = 1Tr(ρ)=1。
  2. 任意算符的平均值为 ⟨α⟩=∑mPm⟨m∣α^∣m⟩=Tr(ρα^)\braket{\alpha}=\sum_m P_m \bra{m}\hat{\alpha}\ket{m} = \text{Tr}(\rho \hat{\alpha})⟨α⟩=∑m​Pm​⟨m∣α^∣m⟩=Tr(ρα^)。
  3. 在更为一般的情况下，密度矩阵为

ρ=∑mPm∣m⟩⟨m∣=∑m∑ξ′ξ′′Pm∣ξ′⟩⟨ξ′∣m⟩⟨m∣ξ′′⟩⟨ξ′′∣=∑ξ′ξ′′(∑mPm⟨ξ′∣m⟩⟨m∣ξ′′⟩)∣ξ′⟩⟨ξ′′∣≡∑ξ′ξ′′ρξ′ξ′′∣ξ′⟩⟨ξ′′∣\rho = \sum_m P_m \ket{m}\bra{m} = \sum_m \sum_{\xi'\xi''} P_m \ket{\xi'}\braket{\xi'|m}\braket{m|\xi''}\bra{\xi''} \\[5pt] =\sum_{\xi'\xi''}\left(\sum_m P_m \braket{\xi'|m}\braket{m|\xi''}\right) \ket{\xi'}\bra{\xi''}\equiv \sum_{\xi'\xi''} \rho_{\xi'\xi''} \ket{\xi'}\bra{\xi''}ρ=m∑​Pm​∣m⟩⟨m∣=m∑​ξ′ξ′′∑​Pm​∣ξ′⟩⟨ξ′∣m⟩⟨m∣ξ′′⟩⟨ξ′′∣=ξ′ξ′′∑​(m∑​Pm​⟨ξ′∣m⟩⟨m∣ξ′′⟩)∣ξ′⟩⟨ξ′′∣≡ξ′ξ′′∑​ρξ′ξ′′​∣ξ′⟩⟨ξ′′∣

   其中 ρξ′ξ′′=∑mPm⟨ξ′∣m⟩⟨m∣ξ′′⟩\rho_{\xi'\xi''} = \sum_m P_m \braket{\xi'|m}\braket{m|\xi''}ρξ′ξ′′​=∑m​Pm​⟨ξ′∣m⟩⟨m∣ξ′′⟩ 是密度矩阵在 {∣ξ⟩}\{\ket{\xi}\}{∣ξ⟩} 基下的矩阵元。

ρξ′ξ′′∗=∑mPm⟨ξ′∣m⟩⟨m∣ξ′′⟩∗=∑mPm⟨m∣ξ′′⟩⟨ξ′∣m⟩=ρξ′′ξ′⇒ρ 是厄米矩阵\rho_{\xi'\xi''}^* = \sum_m P_m \braket{\xi'|m}\braket{m|\xi''}^* = \sum_m P_m \braket{m|\xi''}\braket{\xi'|m} = \rho_{\xi''\xi'}\quad \Rightarrow \quad \rho \text{ 是厄米矩阵} ρξ′ξ′′∗​=m∑​Pm​⟨ξ′∣m⟩⟨m∣ξ′′⟩∗=m∑​Pm​⟨m∣ξ′′⟩⟨ξ′∣m⟩=ρξ′′ξ′​⇒ρ 是厄米矩阵

  1. 若某个态的概率 Pm=δmnP_{m}=\delta_{mn}Pm​=δmn​，则密度矩阵化简为纯态的投影算符

ρ=∑mδmn∣m⟩⟨m∣=∣n⟩⟨n∣\rho =\sum_m \delta_{mn} \ket{m}\bra{m} =\ket{n}\bra{n} ρ=m∑​δmn​∣m⟩⟨m∣=∣n⟩⟨n∣

   此时密度矩阵对角元有且仅有一个为 111，其余均为 000。纯态的密度矩阵还满足

ρ2=(∣n⟩⟨n∣)(∣n⟩⟨n∣)=∣n⟩⟨n∣=ρ⇒Tr(ρ2)=Tr(ρ)=1\rho^2 = \left(\ket{n}\bra{n}\right)\left(\ket{n}\bra{n}\right) = \ket{n}\bra{n} = \rho \quad \Rightarrow \quad \text{Tr}(\rho^2) = \text{Tr}(\rho) = 1 ρ2=(∣n⟩⟨n∣)(∣n⟩⟨n∣)=∣n⟩⟨n∣=ρ⇒Tr(ρ2)=Tr(ρ)=1

对于任意态矢 ∣ψ⟩\ket{\psi}∣ψ⟩，其对应的密度矩阵为 ρ=∣ψ⟩⟨ψ∣\rho = \ket{\psi}\bra{\psi}ρ=∣ψ⟩⟨ψ∣，都满足 ρ2=ρ\rho^2 = \rhoρ2=ρ 和 Tr(ρ2)=1\text{Tr}(\rho^2) = 1Tr(ρ2)=1。

  1. 对于更为一般的态，密度矩阵满足

ρ2=(∑mPm∣m⟩⟨m∣)(∑m′Pm′∣m′⟩⟨m′∣)=∑mPm2∣m⟩⟨m∣\rho^2 = \left(\sum_m P_m \ket{m}\bra{m}\right)\left(\sum_{m'} P_{m'} \ket{m'}\bra{m'}\right) = \sum_m P_m^2 \ket{m}\bra{m} ρ2=(m∑​Pm​∣m⟩⟨m∣)(m′∑​Pm′​∣m′⟩⟨m′∣)=m∑​Pm2​∣m⟩⟨m∣

   我们称 Tr(ρ2)\text{Tr}(\rho^2)Tr(ρ2) 为态的纯度。等号仅在纯态时成立，因此可以通过 Tr(ρ2)\text{Tr}(\rho^2)Tr(ρ2) 来判断系统是否处于纯态：若 Tr(ρ2)<1\text{Tr}(\rho^2) < 1Tr(ρ2)<1，则系统处于混合态。

总的来说，一个数学上可接受的密度算符具有以下性质：

  1. 厄米性：ρ^†=ρ^\hat{\rho}^\dagger = \hat{\rho}ρ^​†=ρ^​
  2. 半正定性：⟨ψ∣ρ^∣ψ⟩≥0\bra{\psi}\hat{\rho}\ket{\psi} \geq 0⟨ψ∣ρ^​∣ψ⟩≥0 对任意态矢 ∣ψ⟩\ket{\psi}∣ψ⟩ 成立

注意：经典概率和量子概率是不一样的，例如 ∣ψ⟩=12(∣0⟩+∣1⟩)\ket{\psi} = \frac{1}{\sqrt{2}}(\ket{0} + \ket{1})∣ψ⟩=2​1​(∣0⟩+∣1⟩) 和 ρ=12∣0⟩⟨0∣+12∣1⟩⟨1∣\rho = \frac{1}{2}\ket{0}\bra{0} + \frac{1}{2}\ket{1}\bra{1}ρ=21​∣0⟩⟨0∣+21​∣1⟩⟨1∣ 是不同的。前者是一个纯态，后者是一个混合态。

密度矩阵的凸组合

假设有 nnn 个密度矩阵 {ρ(i)}\left\{\rho^{(i)}\right\}{ρ(i)}，他们都满足

i)ρ(i)=ρ(i)†,ii)Tr(ρ(i))=1,iii)⟨ψ∣ρ(i)∣ψ⟩≥0for all ∣ψ⟩\left. i \right) \quad \rho^{(i)} = \rho^{(i)\dagger}, \quad \left. ii \right) \quad \text{Tr}(\rho^{(i)}) = 1, \quad \left. iii \right) \quad \bra{\psi}\rho^{(i)}\ket{\psi} \geq 0 \quad for \ all \ \ket{\psi} i)ρ(i)=ρ(i)†,ii)Tr(ρ(i))=1,iii)⟨ψ∣ρ(i)∣ψ⟩≥0for all ∣ψ⟩

那么算符 ρ=∑i=1aiρ(i)\rho = \sum_{i=1} a_i \rho^{(i)}ρ=∑i=1​ai​ρ(i) 也满足上述三个性质，其中 0≤ai≤10 \leq a_i \leq 10≤ai​≤1 且 ∑i=1ai=1\sum_{i=1} a_i = 1∑i=1​ai​=1。ρ=∑i=1aiρ(i)\rho = \sum_{i=1} a_i \rho^{(i)}ρ=∑i=1​ai​ρ(i) 也是一个密度矩阵。

所以我们可以说，所有密度矩阵都形成一个凸集（convex set），这样的一个密度矩阵 ρ\rhoρ 称为 {ρ(i)}\left\{\rho^{(i)}\right\}{ρ(i)} 的凸组合（convex combination）。

Lemmay 引理：对于两个密度矩阵 ρ(i)\rho^{(i)}ρ(i) 和 ρ(j)\rho^{(j)}ρ(j)，有 0≤Tr(ρ(i)ρ(j))≤10 \leq \text{Tr}(\rho^{(i)}\rho^{(j)}) \leq 10≤Tr(ρ(i)ρ(j))≤1，其中等号成立当且仅当 ρ(i)=ρ(j)=纯态\rho^{(i)} = \rho^{(j)}=\text{纯态}ρ(i)=ρ(j)=纯态

定理：一个纯态的密度矩阵不能表示为其他非平庸纯态密度矩阵的凸组合；而混态总可以表示为其他非平庸纯态密度矩阵的凸组合。

  • 混态总可以视为纯态的统计混合，这表明纯态比混态更为基本。
  • 把混态写成纯态的凸组合并不是唯一的。

冯·诺依曼熵

衡量一个密度矩阵偏离纯态程度的一个重要量是冯·诺依曼熵，定义为

S=−Tr(ρln⁡ρ)S= -\text{Tr}(\rho \ln \rho) S=−Tr(ρlnρ)

在一般基下，冯诺依曼熵的计算是十分复杂的，但在密度矩阵的本征态基下，计算就非常简单了。设 ρ\rhoρ 的本征值为 {Pm}\{P_m\}{Pm​}，则

ρ=∑mPm∣m⟩⟨m∣⇒ln⁡ρ=∑mln⁡Pm∣m⟩⟨m∣⇒S=∑m−Pmln⁡Pm\rho = \sum_m P_m \ket{m}\bra{m}\quad \Rightarrow \quad \ln \rho = \sum_m \ln P_m \ket{m}\bra{m}\quad \Rightarrow \quad S = \sum_m -P_m \ln P_m ρ=m∑​Pm​∣m⟩⟨m∣⇒lnρ=m∑​lnPm​∣m⟩⟨m∣⇒S=m∑​−Pm​lnPm​

对于一个纯态 S=0S=0S=0，而对于一个混态 S≥0S \geq 0S≥0。当所有本征值相等时，即 Pm=1/NP_m = 1/NPm​=1/N，其中 NNN 是希尔伯特空间的维数，可以证明此时 SSS 取最大值

Smax⁡=−∑m=1N1Nln⁡1N=ln⁡NS_{\max} = - \sum_{m=1}^N \frac{1}{N} \ln \frac{1}{N} = \ln N Smax​=−m=1∑N​N1​lnN1​=lnN

也就是说，冯·诺依曼熵的取值范围为

0≤S≤ln⁡N0 \leq S \leq \ln N 0≤S≤lnN

当 S=0S=0S=0 时，系统处于纯态；当 S=ln⁡NS=\ln NS=lnN 时，系统处于最大混合态。

例：一个热系综的密度矩阵为 ρT=∑Ee−βEZ∣E⟩⟨E∣\rho_T = \sum_E \frac{e^{-\beta E}}{Z} \ket{E}\bra{E}ρT​=∑E​Ze−βE​∣E⟩⟨E∣。当 T→0T \to 0T→0 时，系统处于基态 ∣E0⟩\ket{E_0}∣E0​⟩，此时 ρT=∣E0⟩⟨E0∣\rho_T = \ket{E_0}\bra{E_0}ρT​=∣E0​⟩⟨E0​∣，所以 S=0S=0S=0。当 T→∞T \to \inftyT→∞ 时，e−βE→1e^{-\beta E} \to 1e−βE→1，ρT=1N∑E∣E⟩⟨E∣\rho_T = \frac{1}{N} \sum_{E} \ket{E}\bra{E}ρT​=N1​∑E​∣E⟩⟨E∣，所有能级等概率占据，S=ln⁡NS=\ln NS=lnN。所以最大混合态也叫做无限温度态。

密度矩阵的决定

我们之前提到，一个厄米矩阵 ρ\rhoρ 有 N2N^2N2 个独立实参数（NNN 是希尔伯特空间的维数）。而密度矩阵还要满足 Tr(ρ)=1\text{Tr}(\rho)=1Tr(ρ)=1，所以密度矩阵有 N2−1N^2-1N2−1 个独立实参数。那么什么样的测量可以决定一个 N2−1N^2-1N2−1 个参数的密度矩阵呢？

在一个任意的基 {∣ξ⟩}\{\ket{\xi}\}{∣ξ⟩} 下，密度矩阵有以下形式
