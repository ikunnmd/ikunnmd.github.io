量子测量

哥本哈根学派对量子力学的诠释包含两个过程

  1. 幺正的量子态演化：量子态 ∣ψ⟩\ket{\psi}∣ψ⟩ 随时间演化遵循薛定谔方程

    iℏddt∣ψ⟩=H^∣ψ⟩i\hbar \frac{d}{dt} \ket{\psi} = \hat{H} \ket{\psi} iℏdtd​∣ψ⟩=H^∣ψ⟩

  2. 非幺正的量子测量：当对量子系统进行测量时，量子态会发生波包坍缩（无法用薛定谔方程描述，量子力学不完备的根源）。

冯诺依曼测量理论：在被测系统与仪器发生相互作用后，系统状态会变成纠缠态（幺正过程）。由于仪器是宏观的，它的坍缩会引发被测系统的“坍缩”。也可由以下两个过程描述：
1.1.1.幺正过程：系统与测量仪器发生相互作用后，形成纠缠态。若 SSS 和 DDD 分别为系统和仪器的状态，则有

U∣Sk⟩⊗∣D⟩=∣Sk⟩⊗∣Dk⟩U\ket{S_k} \otimes \ket{D} = \ket{S_k} \otimes \ket{D_k} U∣Sk​⟩⊗∣D⟩=∣Sk​⟩⊗∣Dk​⟩

其中 UUU 是一个测量操作，使得原本无相互作用的系统和仪器发生耦合，形成纠缠态。整个系统的状态变化为

∣ψi⟩=(∑kck∣Sk⟩)⊗∣D⟩⏟直积态→U∣ψf⟩=U∣ψi⟩=∑kck∣Sk⟩⊗∣Dk⟩⏟纠缠态\ket{\psi_i} = \underbrace{\left(\sum_k c_k \ket{S_k}\right) \otimes \ket{D}}_{\text{直积态}} \xrightarrow{U} \ket{\psi_f} =U\ket{\psi_i} = \underbrace{\sum_k c_k \ket{S_k} \otimes \ket{D_k}}_{\text{纠缠态}} ∣ψi​⟩=直积态(k∑​ck​∣Sk​⟩)⊗∣D⟩​​U​∣ψf​⟩=U∣ψi​⟩=纠缠态k∑​ck​∣Sk​⟩⊗∣Dk​⟩​​

∣ψf⟩=∑kck∣Sk⟩⊗∣Dk⟩→坍缩∣Sk⟩⊗∣Dk⟩\ket{\psi_f} = \sum_k c_k \ket{S_k} \otimes \ket{D_k} \xrightarrow{\text{坍缩}} \ket{S_k} \otimes \ket{D_k} ∣ψf​⟩=k∑​ck​∣Sk​⟩⊗∣Dk​⟩坍缩​∣Sk​⟩⊗∣Dk​⟩

我们可以从测量仪器的状态 ∣Dk⟩\ket{D_k}∣Dk​⟩ 来读取系统的状态 ∣Sk⟩\ket{S_k}∣Sk​⟩。

冯诺依曼测量的局限性：仍然存在一个非幺正过程，无法用薛定谔方程描述

完美测量：从仪器状态 ∣Dk⟩\ket{D_k}∣Dk​⟩ 可以唯一地读取系统状态 ∣Sk⟩\ket{S_k}∣Sk​⟩，那么需要以下条件：

  1. 测量不能对系统产生扰动
  2. 仪器态和系统态一一对应

我们总有 ⟨ψi∣ψi⟩=⟨ψf∣ψf⟩\braket{\psi_i | \psi_i}=\braket{\psi_f | \psi_f}⟨ψi​∣ψi​⟩=⟨ψf​∣ψf​⟩，展开得到

量子测量和退相干

波包坍缩：如果对 A^\hat{A}A^ 进行一次测量后的得到结果是 ana_nan​，则第二次测量会得到相同的结果。

坍缩本质上就是一个退相干过程，即本来处于叠加态的 ∣ψ⟩\ket{\psi}∣ψ⟩，变成了某个单独的本征态 ∣an⟩\ket{a_n}∣an​⟩，丧失了相干性，我们称为退相干。这在密度矩阵上体现得尤为直观

ρ=∣ψ⟩⟨ψ∣=∑n,mcncm∗∣an⟩⟨am∣\rho = \ket{\psi}\bra{\psi} = \sum_{n,m} c_n c_m^* \ket{a_n}\bra{a_m} ρ=∣ψ⟩⟨ψ∣=n,m∑​cn​cm∗​∣an​⟩⟨am​∣

测量后，密度矩阵变为

ρM=∑n∣cn∣2∣an⟩⟨an∣\rho_M = \sum_n |c_n|^2 \ket{a_n}\bra{a_n} ρM​=n∑​∣cn​∣2∣an​⟩⟨an​∣

密度矩阵的对角元表示系统处于某个本征态的概率，非对角元表示不同本征态之间的相干性。测量后，非对角元变为零，系统丧失了相干性。

杨氏双缝干射实验就是一个典型的退相干过程。电子通过双缝后，波函数变为

∣ψ⟩=12(∣ψ1⟩+∣ψ2⟩)\ket{\psi} = \frac{1}{\sqrt{2}} \left(\ket{\psi_1} + \ket{\psi_2}\right) ∣ψ⟩=2​1​(∣ψ1​⟩+∣ψ2​⟩)

其中 ∣ψ1⟩\ket{\psi_1}∣ψ1​⟩ 和 ∣ψ2⟩\ket{\psi_2}∣ψ2​⟩ 分别表示电子通过左缝和右缝的波函数。密度矩阵的非对角元 ∣ψ1⟩⟨ψ2∣\ket{\psi_1}\bra{\psi_2}∣ψ1​⟩⟨ψ2​∣ 和 ∣ψ2⟩⟨ψ1∣\ket{\psi_2}\bra{\psi_1}∣ψ2​⟩⟨ψ1​∣ 代表了两条路径之间的相干性。当我们不对电子进行测量时，这些非对角元存在，电子在屏幕上形成干涉条纹。
然而，如果我们在双缝处放置一个测量装置，试图确定电子通过哪一条缝，那么测量过程会引入退相干效应。测量后，密度矩阵变为

ρM=12(∣ψ1⟩⟨ψ1∣+∣ψ2⟩⟨ψ2∣)\rho_M = \frac{1}{2} \left(\ket{\psi_1}\bra{\psi_1} + \ket{\psi_2}\bra{\psi_2}\right) ρM​=21​(∣ψ1​⟩⟨ψ1​∣+∣ψ2​⟩⟨ψ2​∣)

对于一个系综，对 A^\hat{A}A^ 的二次测量同样给出相同的结果。由于大量量子态副本的平均值等同于对系综的测量，且有

弱测量

假设整体初态为 ∣ψi⟩∣ϕ⟩\ket{\psi_i} \ket{\phi}∣ψi​⟩∣ϕ⟩，其中 ∣ψi⟩\ket{\psi_i}∣ψi​⟩ 为系统的初态，∣ϕ⟩\ket{\phi}∣ϕ⟩ 为测量仪器的初态。经过弱测量后，整体态变为

∣ψi⟩∣ϕ⟩→g(t)p^A^∑f∣ψf⟩∣ϕf⟩\ket{\psi_i} \ket{\phi} \xrightarrow{g\left(t\right)\hat{p}\hat{A}} \sum_f \ket{\psi_f} \ket{\phi_f} ∣ψi​⟩∣ϕ⟩g(t)p^​A^​f∑​∣ψf​⟩∣ϕf​⟩

系统态 ∣ψf⟩\ket{\psi_f}∣ψf​⟩ 和测量仪器态 ∣ϕf⟩\ket{\phi_f}∣ϕf​⟩ 之间存在纠缠。若对系统进行后选择，也就是使用投影算符 Πf=∣ϕf⟩⟨ϕf∣\Pi_f = \ket{\phi_f}\bra{\phi_f}Πf​=∣ϕf​⟩⟨ϕf​∣，使整体纠缠态坍缩至某一直积态。使用时间演化算符对整体态进行演化并用 ∣ψf⟩\ket{\psi_f}∣ψf​⟩ 做内积，就是测量后得到 ∣ψf⟩\ket{\psi_f}∣ψf​⟩ 的概率

⟨ψf∣e−iHdt∣ψi⟩∣ϕ⟩=⟨ψf∣e−iγp^A^∣ψi⟩∣ϕ⟩=⟨ψf∣(1−iγp^A^)∣ψi⟩∣ϕ⟩=⟨ψf∣ψi⟩(1−iγAwp^)∣ϕ⟩≈⟨ψf∣ψi⟩e−iγAwp^∣ϕ⟩ \begin{aligned} \bra{\psi_f} e^{-i H dt} \ket{\psi_i} \ket{\phi} &= \bra{\psi_f} e^{-{i} \gamma \hat{p} \hat{A}} \ket{\psi_i} \ket{\phi} = \bra{\psi_f} \left(1 - i \gamma \hat{p} \hat{A} \right) \ket{\psi_i} \ket{\phi} \\[5pt] &= \bra{\psi_f} \psi_i \rangle \left(1 - i \gamma A_w \hat{p} \right) \ket{\phi} \\[5pt] &\approx \bra{\psi_f} \psi_i \rangle e^{-i \gamma A_w \hat{p}} \ket{\phi} \end{aligned} ⟨ψf​∣e−iHdt∣ψi​⟩∣ϕ⟩​=⟨ψf​∣e−iγp^​A^∣ψi​⟩∣ϕ⟩=⟨ψf​∣(1−iγp^​A^)∣ψi​⟩∣ϕ⟩=⟨ψf​∣ψi​⟩(1−iγAw​p^​)∣ϕ⟩≈⟨ψf​∣ψi​⟩e−iγAw​p^​∣ϕ⟩​

式中 AwA_wAw​ 称为可观测量 A^\hat{A}A^ 的弱值

Aw=⟨ψf∣A^∣ψi⟩⟨ψf∣ψi⟩A_w = \frac{\bra{\psi_f} \hat{A} \ket{\psi_i}}{\bra{\psi_f} \psi_i \rangle} Aw​=⟨ψf​∣ψi​⟩⟨ψf​∣A^∣ψi​⟩​

测量后，相当于 e−iγAwp^e^{-i \gamma A_w \hat{p}}e−iγAw​p^​ 作用在测量仪器的初态 ∣ϕ⟩\ket{\phi}∣ϕ⟩ 上。若测量仪器的初态为 ∣ϕ⟩\ket{\phi}∣ϕ⟩，则有

e−iγAwp^∣ϕ⟩=∫−∞+∞dx∣x⟩∣x+γAw⟩∣ϕ⟩e^{-i \gamma A_w \hat{p}} \ket{\phi} =\int_{-\infty}^{+\infty} dx \ket{x} \ket{x + \gamma A_w} \ket{\phi} e−iγAw​p^​∣ϕ⟩=∫−∞+∞​dx∣x⟩∣x+γAw​⟩∣ϕ⟩

假设初态高斯波包的中心位置处为 000

⟨x∣ϕ⟩=ϕ(x)=1(2πσ2)1/4e−x2/4σ2\braket{x|\phi} = \phi(x) = \frac{1}{(2\pi \sigma^2)^{1/4}} e^{-x^2/4\sigma^2} ⟨x∣ϕ⟩=ϕ(x)=(2πσ2)1/41​e−x2/4σ2

测量后，波包中心位移了 γAw\gamma A_wγAw​，即

ϕ(x)→⟨x+γAw∣ϕ⟩=ϕ(x+γAw)=1(2πσ2)1/4e−(x+γAw)2/4σ2\phi(x) \rightarrow \braket{x+\gamma A_w|\phi}=\phi(x + \gamma A_w) = \frac{1}{(2\pi \sigma^2)^{1/4}} e^{-(x + \gamma A_w)^2/4\sigma^2} ϕ(x)→⟨x+γAw​∣ϕ⟩=ϕ(x+γAw​)=(2πσ2)1/41​e−(x+γAw​)2/4σ2

我们称算符 e−iγAwp^e^{-i \gamma A_w \hat{p}}e−iγAw​p^​ 就是一个平移算符。也就是说，AwA_wAw​ 越大，测量仪器的波包位移越大，测量结果越明显，使得弱测量能有更精确的读出，因此 弱值 AwA_wAw​ 是弱测量的一个重要参数。若是末态和初态的内积 ⟨ψf∣ψi⟩\bra{\psi_f} \psi_i \rangle⟨ψf​∣ψi​⟩ 近似正交，也就是趋近于零，则弱值 AwA_wAw​ 会变得非常大，这样通过弱相互作用，我们反而能得到放大的测量信号。

弱测量的核心思想是通过弱耦合和后选择，放大测量信号，从而在不显著扰动系统的情况下，提取有用的信息。相比于测量，弱测量更像是一种放大的手段。

广义量子测量

一组测量算符 {Mm}\{M_m\}{Mm​} 作用在态矢 ∣ψi⟩\ket{\psi_i}∣ψi​⟩ 上，得到测量结果 mmm 的概率为

p(m)=⟨ψi∣Mm†Mm∣ψi⟩p(m) = \bra{\psi_i} M_m^{\dagger} M_m \ket{\psi_i} p(m)=⟨ψi​∣Mm†​Mm​∣ψi​⟩

测量后，系统态变为

∣ψf⟩=Mm∣ψi⟩⟨ψi∣Mm†Mm∣ψi⟩\ket{\psi_f} = \frac{M_m \ket{\psi_i}}{\sqrt{\bra{\psi_i} M_m^{\dagger} M_m \ket{\psi_i}}} ∣ψf​⟩=⟨ψi​∣Mm†​Mm​∣ψi​⟩​Mm​∣ψi​⟩​

测量算符需满足完备性关系

∑mMm†Mm=I⇒∑m⟨ψi∣Mm†Mm∣ψi⟩=∑mp(m)=1\sum_m M_m^{\dagger} M_m = I \Rightarrow \sum_m \bra{\psi_i} M_m^{\dagger} M_m \ket{\psi_i} = \sum_m p(m) = 1 m∑​Mm†​Mm​=I⇒m∑​⟨ψi​∣Mm†​Mm​∣ψi​⟩=m∑​p(m)=1

  • MmM_mMm​ 不一定是厄米算符
  • 根据Neumark’s theorem，广义量子测量可以通过幺正变换和更大的希尔伯特空间内的投影测量来实现。

投影测量就是一种最简单的广义量子测量：将任意态用一组正交完备基展开 ∣ψi⟩=∑mcm∣m⟩\ket{\psi_i} = \sum_m c_m \ket{m}∣ψi​⟩=∑m​cm​∣m⟩，并定义投影算符 Mm=∣m⟩⟨m∣M_m = \ket{m}\bra{m}Mm​=∣m⟩⟨m∣，则测量结果为 mmm 的概率为

p(m)=⟨ψi∣Mm†Mm∣ψi⟩=⟨ψi∣Mm∣ψi⟩=∣cm∣2p(m) = \bra{\psi_i} M_m^{\dagger} M_m \ket{\psi_i} = \bra{\psi_i} M_m \ket{\psi_i} = |c_m|^2 p(m)=⟨ψi​∣Mm†​Mm​∣ψi​⟩=⟨ψi​∣Mm​∣ψi​⟩=∣cm​∣2

投影测量要求基矢为正交完备基，投影算符是厄米算符

下面介绍另一种广义量子测量：POVM（Positive Operator-Valued Measure，正定算符值测量）。POVM 由一组正定算符 {Fm}\{F_m\}{Fm​} 组成，满足完备性关系

Fm=Mm†Mm,∑mFm=IF_m=M_m^\dagger M_m,\quad \sum_m F_m = I Fm​=Mm†​Mm​,m∑​Fm​=I

测量后被测态被投影至投影态 ∣Mm⟩\ket{M_m}∣Mm​⟩ 的概率为

p(m)=⟨ψi∣Fm∣ψi⟩=⟨ψi∣Mm†Mm∣ψi⟩p(m) = \bra{\psi_i} F_m \ket{\psi_i} = \bra{\psi_i} M_m^\dagger M_m \ket{\psi_i} p(m)=⟨ψi​∣Fm​∣ψi​⟩=⟨ψi​∣Mm†​Mm​∣ψi​⟩

可在一定程度上区分非正交态。

量子不可克隆定理

假设存在一个量子克隆机 UUU，由于我们要得到一个确定性的结果，那么 UUU 须是幺正算符（若是投影测量则不确定），任意态 ∣ψ⟩\ket{\psi}∣ψ⟩ 都能被克隆为两份 ∣ψ⟩∣ψ⟩\ket{\psi}\ket{\psi}∣ψ⟩∣ψ⟩，即

U∣ψ⟩∣e⟩=∣ψ⟩∣ψ⟩U\ket{\psi}\ket{e} = \ket{\psi}\ket{\psi} U∣ψ⟩∣e⟩=∣ψ⟩∣ψ⟩

其中 ∣e⟩\ket{e}∣e⟩ 为初始态。考虑两个不同的态 ∣ψ1⟩\ket{\psi_1}∣ψ1​⟩ 和 ∣ψ2⟩\ket{\psi_2}∣ψ2​⟩，则有

U∣ψ1⟩A∣e⟩B=eiα(ψ1,e)∣ψ1⟩A∣ψ1⟩BU\ket{\psi_1}_A\ket{e}_B =e^{i\alpha\left(\psi_1,e\right)} \ket{\psi_1}_A\ket{\psi_1}_B U∣ψ1​⟩A​∣e⟩B​=eiα(ψ1​,e)∣ψ1​⟩A​∣ψ1​⟩B​

U∣ψ2⟩A∣e⟩B=eiα(ψ2,e)∣ψ2⟩A∣ψ2⟩BU\ket{\psi_2}_A\ket{e}_B =e^{i\alpha\left(\psi_2,e\right)} \ket{\psi_2}_A\ket{\psi_2}_B U∣ψ2​⟩A​∣e⟩B​=eiα(ψ2​,e)∣ψ2​⟩A​∣ψ2​⟩B​

下标 ABABAB 表示原始态和被复制的态，对两式做内积得到

⟨ψ2∣ψ1⟩=⟨ψ2∣ψ1⟩2\braket{\psi_2 | \psi_1} = \braket{\psi_2 | \psi_1}^2 ⟨ψ2​∣ψ1​⟩=⟨ψ2​∣ψ1​⟩2

解得 ⟨ψ2∣ψ1⟩=0\braket{\psi_2 | \psi_1} = 0⟨ψ2​∣ψ1​⟩=0 或 111，即 ∣ψ1⟩\ket{\psi_1}∣ψ1​⟩ 和 ∣ψ2⟩\ket{\psi_2}∣ψ2​⟩ 必须是正交态或相同态，才能被克隆。

  • 量子不可克隆定理说明了，无法构造一个通用的量子克隆机来复制任意未知的量子态。
  • CNOT 门可以克隆两个正交态 ∣0⟩\ket{0}∣0⟩ 和 ∣1⟩\ket{1}∣1⟩，但无法克隆任意态 ∣ψ⟩=α∣0⟩+β∣1⟩\ket{\psi} = \alpha\ket{0} + \beta\ket{1}∣ψ⟩=α∣0⟩+β∣1⟩。
  • 可以通过牺牲克隆的确定性，来实现两个任意态之间的概率性克隆。
