量子信息笔记（三）：量子信息中的基础量子力学

量子力学基本假设

假设一：量子系统的状态由希尔伯特空间 H\mathcal{H}H 内的矢量 ∣ψ⟩\ket{\psi}∣ψ⟩ 描述，其演化由薛定谔方程描述

iℏddt∣ψ⟩=H^∣ψ⟩i\hbar \frac{d}{dt} \ket{\psi} = \hat{H} \ket{\psi} iℏdtd​∣ψ⟩=H^∣ψ⟩

假设二：可观测量 AAA 由希尔伯特空间 H\mathcal{H}H 内的算符 A^\hat{A}A^ 描述，其测量结果为 A^\hat{A}A^ 的本征值。希尔伯特空间内的任意态 ∣ψ⟩\ket{\psi}∣ψ⟩ 都可以由 ∣an⟩\ket{a_n}∣an​⟩ 线性叠加得到

∣ψ⟩=∑ncn∣an⟩\ket{\psi} = \sum_n c_n \ket{a_n} ∣ψ⟩=n∑​cn​∣an​⟩

观测到的结果为 A^\hat{A}A^ 的本征值 ana_nan​，对应的概率为

P(an)=∣⟨an∣ψ⟩∣2=∣cn∣2P(a_n) = \left|\langle a_n | \psi \rangle\right|^2 =\left|c_n\right|^2 P(an​)=∣⟨an​∣ψ⟩∣2=∣cn​∣2

多次测量的平均值为

⟨A^⟩=⟨ψ∣A^∣ψ⟩=∑n∣cn∣2an\langle \hat{A} \rangle = \bra{\psi} \hat{A} \ket{\psi} = \sum_n |c_n|^2 a_n ⟨A^⟩=⟨ψ∣A^∣ψ⟩=n∑​∣cn​∣2an​

假设三：对于一个系统的态 ∣ψ⟩\ket{\psi}∣ψ⟩，如果在测量 A^\hat{A}A^ 后得到本征值 ana_nan​，则系统的态会发生坍缩，变为对应的本征态 ∣an⟩\ket{a_n}∣an​⟩。

投影测量=波包塌缩=波函数约化

直积空间：两个希尔伯特空间 H1\mathcal{H}_1H1​ 和 H2\mathcal{H}_2H2​ 的维数分别为 n1n_1n1​ 和 n2n_2n2​；态矢分别为 ∣ψi⟩∈H1\ket{\psi_i} \in \mathcal{H}_1∣ψi​⟩∈H1​ 和 ∣ψi⟩∈H2\ket{\psi_i} \in \mathcal{H}_2∣ψi​⟩∈H2​，i=1,2…i=1,2 \dotsi=1,2…；A^i\hat{A}_iA^i​ 和 B^i\hat{B}_iB^i​ 分别是 H1\mathcal{H}_1H1​ 和 H2\mathcal{H}_2H2​ 上的线性算符。则它们的直积空间为

H=H1⊗H2 dim=n1n2\mathcal{H}=\mathcal{H}_1 \otimes \mathcal{H}_2 \qquad\text{ dim} = n_1 n_2 H=H1​⊗H2​ dim=n1​n2​

直积态 ∣ψi⟩⊗∣ϕj⟩≡∣ψi⟩∣ϕj⟩\ket{\psi_i} \otimes \ket{\phi_j} \equiv \ket{\psi_i}\ket{\phi_j}∣ψi​⟩⊗∣ϕj​⟩≡∣ψi​⟩∣ϕj​⟩ 为 H\mathcal{H}H 内的矢量，直积空间中的内积和算符作用满足

(⟨ψi∣⟨ϕj∣)(∣ψi′⟩∣ϕj′⟩)=(⟨ψi∣ψi′)(⟨ϕj∣ϕj′⟩)\left(\bra{\psi_i} \bra{\phi_j}\right) \left(\ket{\psi_i'} \ket{\phi_j'}\right) = \left(\langle \psi_i | \psi_i' \right)\left(\langle \phi_j | \phi_j' \rangle \right) (⟨ψi​∣⟨ϕj​∣)(∣ψi′​⟩∣ϕj′​⟩)=(⟨ψi​∣ψi′​)(⟨ϕj​∣ϕj′​⟩)

(A^⊗B^)(∣ψi⟩∣ϕj⟩)=(A^∣ψi⟩)⊗(B^∣ϕj⟩)(\hat{A} \otimes \hat{B}) (\ket{\psi_i} \ket{\phi_j}) = (\hat{A} \ket{\psi_i}) \otimes (\hat{B} \ket{\phi_j}) (A^⊗B^)(∣ψi​⟩∣ϕj​⟩)=(A^∣ψi​⟩)⊗(B^∣ϕj​⟩)

直积态在整体希尔伯特空间内态矢的内积就相当于，每个希尔伯特子空间内的态矢做内积再相乘

量子测量

哥本哈根学派对量子力学的诠释包含两个过程

  1. 幺正的量子态演化：量子态 ∣ψ⟩\ket{\psi}∣ψ⟩ 随时间演化遵循薛定谔方程

    iℏddt∣ψ⟩=H^∣ψ⟩i\hbar \frac{d}{dt} \ket{\psi} = \hat{H} \ket{\psi} iℏdtd​∣ψ⟩=H^∣ψ⟩

  2. 非幺正的量子测量：当对量子系统进行测量时，量子态会发生波包坍缩（无法用薛定谔方程描述，量子力学不完备的根源）。

冯诺依曼测量理论：在被测系统与仪器发生相互作用后，系统状态会变成纠缠态（幺正过程）。由于仪器是宏观的，它的坍缩会引发被测系统的“坍缩”。也可由以下两个过程描述：
1.1.1.幺正过程：系统与测量仪器发生相互作用后，形成纠缠态。若 SSS 和 DDD 分别为系统和仪器的状态，则有

U∣Sk⟩⊗∣D⟩=∣Sk⟩⊗∣Dk⟩U\ket{S_k} \otimes \ket{D} = \ket{S_k} \otimes \ket{D_k} U∣Sk​⟩⊗∣D⟩=∣Sk​⟩⊗∣Dk​⟩

其中 UUU 是一个测量操作，使得原本无相互作用的系统和仪器发生耦合，形成纠缠态。整个系统的状态变化为

∣ψi⟩=(∑kck∣Sk⟩)⊗∣D⟩⏟直积态→U∣ψf⟩=U∣ψi⟩=∑kck∣Sk⟩⊗∣Dk⟩⏟纠缠态\ket{\psi_i} = \underbrace{\left(\sum_k c_k \ket{S_k}\right) \otimes \ket{D}}_{\text{直积态}} \xrightarrow{U} \ket{\psi_f} =U\ket{\psi_i} = \underbrace{\sum_k c_k \ket{S_k} \otimes \ket{D_k}}_{\text{纠缠态}} ∣ψi​⟩=直积态(k∑​ck​∣Sk​⟩)⊗∣D⟩​​U​∣ψf​⟩=U∣ψi​⟩=纠缠态k∑​ck​∣Sk​⟩⊗∣Dk​⟩​​

2.2.2.非幺正过程：仪器的波包坍缩

∣ψf⟩=∑kck∣Sk⟩⊗∣Dk⟩→坍缩∣Sk⟩⊗∣Dk⟩\ket{\psi_f} = \sum_k c_k \ket{S_k} \otimes \ket{D_k} \xrightarrow{\text{坍缩}} \ket{S_k} \otimes \ket{D_k} ∣ψf​⟩=k∑​ck​∣Sk​⟩⊗∣Dk​⟩坍缩​∣Sk​⟩⊗∣Dk​⟩

我们可以从测量仪器的状态 ∣Dk⟩\ket{D_k}∣Dk​⟩ 来读取系统的状态 ∣Sk⟩\ket{S_k}∣Sk​⟩。

冯诺依曼测量的局限性：仍然存在一个非幺正过程，无法用薛定谔方程描述

完美测量：从仪器状态 ∣Dk⟩\ket{D_k}∣Dk​⟩ 可以唯一地读取系统状态 ∣Sk⟩\ket{S_k}∣Sk​⟩，那么需要以下条件：

  1. 测量不能对系统产生扰动
  2. 仪器态和系统态一一对应

我们总有 ⟨ψi∣ψi⟩=⟨ψf∣ψf⟩\braket{\psi_i | \psi_i}=\braket{\psi_f | \psi_f}⟨ψi​∣ψi​⟩=⟨ψf​∣ψf​⟩，展开得到

⟨Sk∣Sk′⟩(1−⟨Dk∣Dk′⟩)=0\braket{S_k | S_k'} \left(1-\braket{D_k | D_k'}\right) = 0 ⟨Sk​∣Sk′​⟩(1−⟨Dk​∣Dk′​⟩)=0

因此被测态必须满足 ⟨Sk∣Sk′⟩=δkk′\braket{S_k | S_k'} = \delta_{kk'}⟨Sk​∣Sk′​⟩=δkk′​，才能实现确定性测量。

量子测量和退相干

波包坍缩：如果对 A^\hat{A}A^ 进行一次测量后的得到结果是 ana_nan​，则第二次测量会得到相同的结果。

坍缩本质上就是一个退相干过程，即本来处于叠加态的 ∣ψ⟩\ket{\psi}∣ψ⟩，变成了某个单独的本征态 ∣an⟩\ket{a_n}∣an​⟩，丧失了相干性，我们称为退相干。这在密度矩阵上体现得尤为直观

ρ=∣ψ⟩⟨ψ∣=∑n,mcncm∗∣an⟩⟨am∣\rho = \ket{\psi}\bra{\psi} = \sum_{n,m} c_n c_m^* \ket{a_n}\bra{a_m} ρ=∣ψ⟩⟨ψ∣=n,m∑​cn​cm∗​∣an​⟩⟨am​∣

测量后，密度矩阵变为

ρM=∑n∣cn∣2∣an⟩⟨an∣\rho_M = \sum_n |c_n|^2 \ket{a_n}\bra{a_n} ρM​=n∑​∣cn​∣2∣an​⟩⟨an​∣

密度矩阵的对角元表示系统处于某个本征态的概率，非对角元表示不同本征态之间的相干性。测量后，非对角元变为零，系统丧失了相干性。

杨氏双缝干射实验就是一个典型的退相干过程。电子通过双缝后，波函数变为

∣ψ⟩=12(∣ψ1⟩+∣ψ2⟩)\ket{\psi} = \frac{1}{\sqrt{2}} \left(\ket{\psi_1} + \ket{\psi_2}\right) ∣ψ⟩=2​1​(∣ψ1​⟩+∣ψ2​⟩)

密度矩阵为

ρ=∣ψ⟩⟨ψ∣=12(∣ψ1⟩⟨ψ1∣+∣ψ2⟩⟨ψ2∣+∣ψ1⟩⟨ψ2∣+∣ψ2⟩⟨ψ1∣)\rho = \ket{\psi}\bra{\psi} = \frac{1}{2} \left(\ket{\psi_1}\bra{\psi_1} + \ket{\psi_2}\bra{\psi_2} + \ket{\psi_1}\bra{\psi_2} + \ket{\psi_2}\bra{\psi_1}\right) ρ=∣ψ⟩⟨ψ∣=21​(∣ψ1​⟩⟨ψ1​∣+∣ψ2​⟩⟨ψ2​∣+∣ψ1​⟩⟨ψ2​∣+∣ψ2​⟩⟨ψ1​∣)

其中 ∣ψ1⟩\ket{\psi_1}∣ψ1​⟩ 和 ∣ψ2⟩\ket{\psi_2}∣ψ2​⟩ 分别表示电子通过左缝和右缝的波函数。密度矩阵的非对角元 ∣ψ1⟩⟨ψ2∣\ket{\psi_1}\bra{\psi_2}∣ψ1​⟩⟨ψ2​∣ 和 ∣ψ2⟩⟨ψ1∣\ket{\psi_2}\bra{\psi_1}∣ψ2​⟩⟨ψ1​∣ 代表了两条路径之间的相干性。当我们不对电子进行测量时，这些非对角元存在，电子在屏幕上形成干涉条纹。
然而，如果我们在双缝处放置一个测量装置，试图确定电子通过哪一条缝，那么测量过程会引入退相干效应。测量后，密度矩阵变为

ρM=12(∣ψ1⟩⟨ψ1∣+∣ψ2⟩⟨ψ2∣)\rho_M = \frac{1}{2} \left(\ket{\psi_1}\bra{\psi_1} + \ket{\psi_2}\bra{\psi_2}\right) ρM​=21​(∣ψ1​⟩⟨ψ1​∣+∣ψ2​⟩⟨ψ2​∣)

非对角元消失，电子丧失了两条路径之间的相干性，屏幕上不再出现干涉条纹，而是形成两条独立的亮斑。

对于一个系综，对 A^\hat{A}A^ 的二次测量同样给出相同的结果。由于大量量子态副本的平均值等同于对系综的测量，且有

⟨A^⟩=Tr(ρA^)=Tr(ρMA^)\langle \hat{A} \rangle = \text{Tr}(\rho \hat{A}) =\text{Tr}(\rho_M \hat{A}) ⟨A^⟩=Tr(ρA^)=Tr(ρM​A^)

这表明，测量（退相干）与否不改变系综的一个算符 A^\hat{A}A^ 的平均值。

我们还可以看到，测量后的密度矩阵 ρM\rho_MρM​ 已经由一个纯态密度矩阵变成了混态密度矩阵，同时量子概率转变为经典概率。

ρ=∣ψ⟩⟨ψ∣=∑n,mcncm∗∣an⟩⟨am∣→测量ρM=∑n∣cn∣2∣an⟩⟨an∣\rho = \ket{\psi}\bra{\psi} = \sum_{n,m} c_n c_m^* \ket{a_n}\bra{a_m} \xrightarrow{\text{测量}} \rho_M = \sum_n |c_n|^2 \ket{a_n}\bra{a_n} ρ=∣ψ⟩⟨ψ∣=n,m∑​cn​cm∗​∣an​⟩⟨am​∣测量​ρM​=n∑​∣cn​∣2∣an​⟩⟨an​∣

Stern-Gerlach 实验

Stern-Gerlach 实验中，原子自旋态为被测态，磁场梯度和空间条纹为测量仪器，薛定谔方程可写为

iℏ∂∂t∣S(t)⟩∣ψ(t)⟩=HS−G∣S(t)⟩∣ψ(t)⟩i\hbar \frac{\partial}{\partial t} \ket{S\left(t\right)}\ket{\psi\left(t\right)} = H_{S-G}\ket{S\left(t\right)}\ket{\psi\left(t\right)} iℏ∂t∂​∣S(t)⟩∣ψ(t)⟩=HS−G​∣S(t)⟩∣ψ(t)⟩

其中 ∣S(t)⟩=c↑∣↑⟩+c↓∣↓⟩\ket{S\left(t\right)}=c_{\uparrow}\ket{\uparrow} + c_{\downarrow}\ket{\downarrow}∣S(t)⟩=c↑​∣↑⟩+c↓​∣↓⟩ 为待测系统； ∣ψ(t)⟩\ket{\psi\left(t\right)}∣ψ(t)⟩ 为空间波函数，起到仪器的作用。
现在我们讨论为什么要强调 Stern-Gerlach 装置的非匀强磁场。假设磁场是匀强的，则哈密顿量为

HS−G=p^22m−gσ^⋅B⃗H_{S-G}=\frac{\hat{p}^2}{2m}-g \hat{\sigma} \cdot \vec{B} HS−G​=2mp^​2​−gσ^⋅B

薛定谔方程变为

iℏ∂∂t∣S(t)⟩∣ψ(t)⟩=(p^22m−gσ^⋅B⃗)∣S(t)⟩∣ψ(t)⟩i\hbar \frac{\partial}{\partial t} \ket{S\left(t\right)}\ket{\psi\left(t\right)} = \left(\frac{\hat{p}^2}{2m}-g \hat{\sigma} \cdot \vec{B}\right)\ket{S\left(t\right)}\ket{\psi\left(t\right)} iℏ∂t∂​∣S(t)⟩∣ψ(t)⟩=(2mp^​2​−gσ^⋅B)∣S(t)⟩∣ψ(t)⟩

若磁场为匀强磁场，则哈密顿量中的 p^22m\frac{\hat{p}^2}{2m}2mp^​2​ 单独作用于波函数 ∣ψ(t)⟩\ket{\psi\left(t\right)}∣ψ(t)⟩，−gσ^⋅B⃗-g \hat{\sigma} \cdot \vec{B}−gσ^⋅B 单独作用于自旋态 ∣S(t)⟩\ket{S\left(t\right)}∣S(t)⟩，因此系统的态矢仍然是一个直积态。只有当磁场为非匀强磁场时，哈密顿量为

HS−G=p^22m−gσ^⋅B⃗(r^)H_{S-G}=\frac{\hat{p}^2}{2m}-g \hat{\sigma} \cdot \vec{B}\left(\hat{r}\right) HS−G​=2mp^​2​−gσ^⋅B(r^)

此时自旋-磁场耦合项 −gσ^⋅B⃗(r^)-g\hat{\sigma} \cdot \vec{B}\left(\hat{r}\right)−gσ^⋅B(r^) 同时作用于 ∣S(t)⟩\ket{S\left(t\right)}∣S(t)⟩ 和 ∣ψ(t)⟩\ket{\psi\left(t\right)}∣ψ(t)⟩，使得自旋态和空间波函数之间产生了纠缠，满足了测量的要求。
