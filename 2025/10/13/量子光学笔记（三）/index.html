量子光学笔记（三）：相干态

发表于2025-10-13|更新于2025-10-14|课堂笔记
|总字数:2.3k|阅读时长:11分钟|浏览量:

我们为什么要研究相干态？相干态是联系经典光学与量子光学的桥梁，例如量子描述中的平面波可以用相干态来表示。

湮灭算符的本征态

假设湮灭算符的本征态为 ∣α⟩|\alpha\rangle∣α⟩，满足

a^∣α⟩=α∣α⟩\hat{a}|\alpha\rangle = \alpha |\alpha\rangle a^∣α⟩=α∣α⟩

将 ∣α⟩|\alpha\rangle∣α⟩ 展开在 Fock 态基底上∣α⟩=∑n=0∞bn∣n⟩|\alpha\rangle = \sum_{n=0}^{\infty} b_n |n\rangle∣α⟩=∑n=0∞​bn​∣n⟩，代入上式得

a^∣α⟩=∑n=0∞bna^∣n⟩=∑n=1∞bnn∣n−1⟩=∑n=0∞bn+1n+1∣n⟩=∑n=0∞bnα∣n⟩\hat{a}\ket{\alpha} = \sum_{n=0}^{\infty} b_n \hat{a}|n\rangle = \sum_{n=1}^{\infty} b_n \sqrt{n}|n-1\rangle=\sum_{n=0}^{\infty} b_{n+1} \sqrt{n+1}|n\rangle=\sum_{n=0}^{\infty} b_n \alpha|n\rangle a^∣α⟩=n=0∑∞​bn​a^∣n⟩=n=1∑∞​bn​n​∣n−1⟩=n=0∑∞​bn+1​n+1​∣n⟩=n=0∑∞​bn​α∣n⟩

解出递推关系

bn=αnn!b0b_n = \frac{\alpha^n}{\sqrt{n!}} b_0 bn​=n!​αn​b0​

因此，∣α⟩|\alpha\rangle∣α⟩ 可以写成

∣α⟩=∑n=0∞bn∣n⟩=b0∑n=0∞αnn!∣n⟩|\alpha\rangle = \sum_{n=0}^{\infty} b_n |n\rangle = b_0 \sum_{n=0}^{\infty} \frac{\alpha^n}{\sqrt{n!}} |n\rangle ∣α⟩=n=0∑∞​bn​∣n⟩=b0​n=0∑∞​n!​αn​∣n⟩

归一化条件 ⟨α∣α⟩=1\langle \alpha|\alpha\rangle = 1⟨α∣α⟩=1 及 eee 指数的泰勒展开给出

∣b0∣2∑n=0∞∣α∣2nn!=∣b0∣2e∣α∣2=1|b_0|^2 \sum_{n=0}^{\infty} \frac{|\alpha|^{2n}}{n!} = |b_0|^2 e^{|\alpha|^2} = 1 ∣b0​∣2n=0∑∞​n!∣α∣2n​=∣b0​∣2e∣α∣2=1

因此 b0=e−∣α∣2/2b_0 = e^{-|\alpha|^2/2}b0​=e−∣α∣2/2，湮灭算符本征态的最终表达式为

∣α⟩=e−12∣α∣2∑n=0∞αnn!∣n⟩|\alpha\rangle = e^{-\frac{1}{2}|\alpha|^2} \sum_{n=0}^{\infty} \frac{\alpha^n}{\sqrt{n!}} |n\rangle ∣α⟩=e−21​∣α∣2n=0∑∞​n!​αn​∣n⟩

我们将湮灭算符的本征态称为相干态。

相干态的波函数

利用 ⟨x∣(a^−α)∣α⟩=0\bra{x}\left(\hat{a}-\alpha\right)|\alpha\rangle = 0⟨x∣(a^−α)∣α⟩=0，可以求出相干态在位置表象下的波函数 ψα(x)=⟨x∣α⟩\psi_{\alpha}(x) = \langle x|\alpha\rangleψα​(x)=⟨x∣α⟩：我们已知

a^=mω2ℏ(x^+imωp^)=mω2ℏ(x+ℏmω∂∂x)=12xˉ0(x+xˉ02∂∂x)\hat{a} =\sqrt{\frac{m\omega}{2\hbar}}\left(\hat{x} + \frac{i}{m\omega}\hat{p}\right) = \sqrt{\frac{m\omega}{2\hbar}}\left(x + \frac{\hbar}{m\omega}\frac{\partial}{\partial x}\right) = \frac{1}{\sqrt{2}\bar{x}_0}\left(x + \bar{x}_0^2 \frac{\partial}{\partial x}\right) a^=2ℏmω​​(x^+mωi​p^​)=2ℏmω​​(x+mωℏ​∂x∂​)=2​xˉ0​1​(x+xˉ02​∂x∂​)

其中 xˉ02=ℏmω\bar{x}_0^2 = \frac{\hbar}{m\omega}xˉ02​=mωℏ​，代入上式得

[12xˉ0(x+xˉ02∂∂x)−α]ψα(x)=0\left[\frac{1}{\sqrt{2\bar{x}_0}}\left(x + \bar{x}_0^2 \frac{\partial}{\partial x}\right)-\alpha\right]\psi_{\alpha}(x) = 0 [2xˉ0​​1​(x+xˉ02​∂x∂​)−α]ψα​(x)=0

[(x−2xˉ0α)+xˉ02∂∂x]ψα(x)=0\left[\left(x-\sqrt{2}\bar{x}_0 \alpha\right) + \bar{x}_0^2 \frac{\partial}{\partial x}\right]\psi_{\alpha}(x) = 0 [(x−2​xˉ0​α)+xˉ02​∂x∂​]ψα​(x)=0

解出波函数

ψα(x)=⟨x∣α⟩∝exp⁡[−(x−2xˉ0α)22xˉ02]\psi_{\alpha}(x) =\braket{x|\alpha} \propto \exp\left[-\frac{(x-\sqrt{2}\bar{x}_0 \alpha)^2}{2\bar{x}_0^2}\right] ψα​(x)=⟨x∣α⟩∝exp[−2xˉ02​(x−2​xˉ0​α)2​]

对比基态波函数

ψ0(x)=⟨x∣0⟩∝exp⁡[−x22xˉ02]\psi_0(x) =\braket{x|0} \propto \exp\left[-\frac{x^2}{2\bar{x}_0^2}\right] ψ0​(x)=⟨x∣0⟩∝exp[−2xˉ02​x2​]

我们发现相干态波函数和基态波函数都是高斯波包，只不过相干态的波包中心被平移到了 x=2xˉ0αx=\sqrt{2}\bar{x}_0 \alphax=2​xˉ0​α 处。

注意：这里的 α\alphaα 是复数，因此相干态波函数的中心不是在空间的简单移动。

相干态的性质

非正交性：相干态不是正交归一的，两个不同的相干态的内积为

⟨α∣β⟩=e−12(∣α∣2+∣β∣2)∑n=0∞α∗nn!βnn!=e−12(∣α∣2+∣β∣2−2α∗β)\langle \alpha|\beta\rangle =e^{-\frac{1}{2}(|\alpha|^2 + |\beta|^2)}\sum_{n=0}^{\infty}\frac{\alpha^{*n}}{\sqrt{n!}}\frac{\beta^{n}}{\sqrt{n!}}=e^{-\frac{1}{2}(|\alpha|^2 + |\beta|^2 - 2\alpha^*\beta)} ⟨α∣β⟩=e−21​(∣α∣2+∣β∣2)n=0∑∞​n!​α∗n​n!​βn​=e−21​(∣α∣2+∣β∣2−2α∗β)

当 ∣α−β∣|\alpha - \beta|∣α−β∣ 很大时，也就是说 α\alphaα 和 β\betaβ 两个点在复平面内距离很远时，⟨α∣β⟩≈0\langle \alpha|\beta\rangle \approx 0⟨α∣β⟩≈0，此时两个相干态近似正交。

过完备性：由于相干态不是正交的，因此不满足一般的完备性关系 ∑n∣n⟩⟨n∣=I\sum_n\ket{n}\bra{n} = \mathrm{I}∑n​∣n⟩⟨n∣=I，但是可以证明，相干态满足过完备关系

∫d2απ∣α⟩⟨α∣=I\int \frac{d^2\alpha}{\pi} |\alpha\rangle \langle \alpha| = \mathrm{I} ∫πd2α​∣α⟩⟨α∣=I

其中 d2α=d(Reα)d(Imα)d^2\alpha = d(\mathrm{Re}\alpha)d(\mathrm{Im}\alpha)d2α=d(Reα)d(Imα)。

位移算符

使用位移算符也同样可以得到相干态，位移算符定义为

D^α:=eαa^†−α∗a^\hat{D}_{\alpha} := e^{\alpha \hat{a}^{\dagger} - \alpha^* \hat{a}} D^α​:=eαa^†−α∗a^

它是一个幺正算符，满足 D^α†=D^α−1\hat{D}_{\alpha}^{\dagger} = \hat{D}_{\alpha}^{-1}D^α†​=D^α−1​。可以利用 Baker-Campbell-Hausdorff 公式证明 D^α†∣0⟩=∣α⟩\hat{D}_{\alpha}^\dagger \ket{0} = \ket{\alpha}D^α†​∣0⟩=∣α⟩

D^α†∣0⟩=e−12∣α∣2eαa^†−α∗a^∣0⟩=e−12∣α∣2∑n=0∞αnn!(a^†)n∣0⟩=e−12∣α∣2∑n=0∞αnn!∣n⟩\hat{D}_{\alpha}^\dagger \ket{0} =e^{-\frac{1}{2}|\alpha|^2}e^{\alpha \hat{a}^{\dagger} - \alpha^* \hat{a}} \ket{0} = e^{-\frac{1}{2}|\alpha|^2} \sum_{n=0}^{\infty} \frac{\alpha^{n}}{n!}\left(\hat{a}^{\dagger}\right)^n |0\rangle= e^{-\frac{1}{2}|\alpha|^2} \sum_{n=0}^{\infty} \frac{\alpha^{n}}{\sqrt{n!}}|n\rangle D^α†​∣0⟩=e−21​∣α∣2eαa^†−α∗a^∣0⟩=e−21​∣α∣2n=0∑∞​n!αn​(a^†)n∣0⟩=e−21​∣α∣2n=0∑∞​n!​αn​∣n⟩

我们还可以注意到 D^α†=D^−α\hat{D}_{\alpha}^{\dagger} = \hat{D}_{-\alpha}D^α†​=D^−α​，因此 D^α\hat{D}_{\alpha}D^α​ 的逆算符为 D^−α\hat{D}_{-\alpha}D^−α​，这就相当于在复平面内移动了 α\alphaα 的距离后再移动 −α-\alpha−α 的距离，回到了原点。但是位移 α\alphaα 的距离后再位移 β\betaβ 的距离，并不等于位移 α+β\alpha + \betaα+β 的距离

D^αD^β=ei Im(αβ∗)D^α+β\hat{D}_{\alpha}\hat{D}_{\beta} = e^{i\ Im\left(\alpha \beta^*\right)}\hat{D}_{\alpha + \beta} D^α​D^β​=ei Im(αβ∗)D^α+β​

Baker-Campbell-Hausdorff 公式

要证明 Baker-Campbell-Hausdorff 公式，首先证明 Baker-Hausdorff 公式

eA^B^e−A^=B^+[A^,B^]+12![A^,[A^,B^]]+13![A^,[A^,[A^,B^]]]+⋯=∑n=0∞1n!(adA^)nB^e^{\hat{A}}\hat{B}e^{-\hat{A}} = \hat{B} + [\hat{A},\hat{B}] + \frac{1}{2!}[\hat{A},[\hat{A},\hat{B}]] + \frac{1}{3!}[\hat{A},[\hat{A},[\hat{A},\hat{B}]]] + \cdots=\sum_{n=0}^{\infty} \frac{1}{n!}\left(ad_{\hat{A}}\right)^n \hat{B} eA^B^e−A^=B^+[A^,B^]+2!1​[A^,[A^,B^]]+3!1​[A^,[A^,[A^,B^]]]+⋯=n=0∑∞​n!1​(adA^​)nB^

其中 adA^ nB^:=[A^,[A^,⋯ ,[A^,B^]⋯ ]⏟n 个 A^ad_{\hat{A}}^{\ n} \hat{B} := \underbrace{[\hat{A},[\hat{A},\cdots,[\hat{A},\hat{B}]\cdots]}_{n \text{ 个 } \hat{A}}adA^ n​B^:=n 个 A^[A^,[A^,⋯,[A^,B^]⋯]​​

在量子力学中，我们经常需要处理形如 O^(t)=U†(t)O^U(t)\hat{O}(t)=U^\dagger\left(t\right) \hat{O} U(t)O^(t)=U†(t)O^U(t) 的幺正变换，特别是在量子态的演化和算符的变换中，Baker-Hausdorff 公式为这提供了便利，例如一个力学量算符在海森堡绘景下的时间演化 O^(t)=eiH^t/ℏO^e−iH^t/ℏ\hat{O}(t) = e^{i\hat{H}t/\hbar} \hat{O} e^{-i\hat{H}t/\hbar}O^(t)=eiH^t/ℏO^e−iH^t/ℏ，对于一个简谐振子

eiH^t/ℏa^e−iH^t/ℏ=eiωta^†a^a^e−iωta^†a^=e−iωta^e^{i\hat{H}t/\hbar} \hat{a} e^{-i\hat{H}t/\hbar} = e^{i\omega t \hat{a}^{\dagger}\hat{a}} \hat{a} e^{-i\omega t\hat{a}^{\dagger}\hat{a} } = e^{-i\omega t}\hat{a} eiH^t/ℏa^e−iH^t/ℏ=eiωta^†a^a^e−iωta^†a^=e−iωta^

D^αa^D^α†=a^−αD^α†a^D^α=a^+α\hat{D}_{\alpha} \hat{a} \hat{D}_{\alpha}^\dagger = \hat{a} - \alpha \\[5pt] \hat{D}_{\alpha}^\dagger \hat{a} \hat{D}_{\alpha} = \hat{a} + \alphaD^α​a^D^α†​=a^−αD^α†​a^D^α​=a^+α

有了 Baker-Hausdorff 公式，就可以证明 Baker-Campbell-Hausdorff 公式：对于两个算符 A^\hat{A}A^ 和 B^\hat{B}B^，如果它们的对易子 [A^,B^][\hat{A},\hat{B}][A^,B^] 与 A^\hat{A}A^ 和 B^\hat{B}B^ 都对易，即 [[A^,B^],A^]=[[A^,B^],B^]=0[[\hat{A},\hat{B}],\hat{A}] = [[\hat{A},\hat{B}],\hat{B}] = 0[[A^,B^],A^]=[[A^,B^],B^]=0，那么有

eA^+B^=e−12[A^,B^]eA^eB^=e12[A^,B^]eB^eA^e^{\hat{A}+\hat{B}} = e^{-\frac{1}{2}[\hat{A},\hat{B}]} e^{\hat{A}} e^{\hat{B}} = e^{\frac{1}{2}[\hat{A},\hat{B}]} e^{\hat{B}} e^{\hat{A}} eA^+B^=e−21​[A^,B^]eA^eB^=e21​[A^,B^]eB^eA^
