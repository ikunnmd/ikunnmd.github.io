量子光学笔记（三）：相干态

我们为什么要研究相干态？相干态是联系经典光学与量子光学的桥梁，例如量子描述中的平面波可以用相干态来表示。

湮灭算符的本征态

假设湮灭算符的本征态为 ∣α⟩|\alpha\rangle∣α⟩，满足

a^∣α⟩=α∣α⟩\hat{a}|\alpha\rangle = \alpha |\alpha\rangle a^∣α⟩=α∣α⟩

将 ∣α⟩|\alpha\rangle∣α⟩ 展开在 Fock 态基底上∣α⟩=∑n=0∞bn∣n⟩|\alpha\rangle = \sum_{n=0}^{\infty} b_n |n\rangle∣α⟩=∑n=0∞​bn​∣n⟩，代入上式得

a^∣α⟩=∑n=0∞bna^∣n⟩=∑n=1∞bnn∣n−1⟩=∑n=0∞bn+1n+1∣n⟩=∑n=0∞bnα∣n⟩\hat{a}\ket{\alpha} = \sum_{n=0}^{\infty} b_n \hat{a}|n\rangle = \sum_{n=1}^{\infty} b_n \sqrt{n}|n-1\rangle=\sum_{n=0}^{\infty} b_{n+1} \sqrt{n+1}|n\rangle=\sum_{n=0}^{\infty} b_n \alpha|n\rangle a^∣α⟩=n=0∑∞​bn​a^∣n⟩=n=1∑∞​bn​n​∣n−1⟩=n=0∑∞​bn+1​n+1​∣n⟩=n=0∑∞​bn​α∣n⟩

解出递推关系

bn=αnn!b0b_n = \frac{\alpha^n}{\sqrt{n!}} b_0 bn​=n!​αn​b0​

因此，∣α⟩|\alpha\rangle∣α⟩ 可以写成

∣α⟩=∑n=0∞bn∣n⟩=b0∑n=0∞αnn!∣n⟩|\alpha\rangle = \sum_{n=0}^{\infty} b_n |n\rangle = b_0 \sum_{n=0}^{\infty} \frac{\alpha^n}{\sqrt{n!}} |n\rangle ∣α⟩=n=0∑∞​bn​∣n⟩=b0​n=0∑∞​n!​αn​∣n⟩

归一化条件 ⟨α∣α⟩=1\langle \alpha|\alpha\rangle = 1⟨α∣α⟩=1 及 eee 指数的泰勒展开给出

∣b0∣2∑n=0∞∣α∣2nn!=∣b0∣2e∣α∣2=1|b_0|^2 \sum_{n=0}^{\infty} \frac{|\alpha|^{2n}}{n!} = |b_0|^2 e^{|\alpha|^2} = 1 ∣b0​∣2n=0∑∞​n!∣α∣2n​=∣b0​∣2e∣α∣2=1

因此 b0=e−∣α∣2/2b_0 = e^{-|\alpha|^2/2}b0​=e−∣α∣2/2，湮灭算符本征态的最终表达式为

∣α⟩=e−12∣α∣2∑n=0∞αnn!∣n⟩|\alpha\rangle = e^{-\frac{1}{2}|\alpha|^2} \sum_{n=0}^{\infty} \frac{\alpha^n}{\sqrt{n!}} |n\rangle ∣α⟩=e−21​∣α∣2n=0∑∞​n!​αn​∣n⟩

我们将湮灭算符的本征态称为相干态。

相干态的波函数

利用 ⟨x∣(a^−α)∣α⟩=0\bra{x}\left(\hat{a}-\alpha\right)|\alpha\rangle = 0⟨x∣(a^−α)∣α⟩=0，可以求出相干态在位置表象下的波函数 ψα(x)=⟨x∣α⟩\psi_{\alpha}(x) = \langle x|\alpha\rangleψα​(x)=⟨x∣α⟩：我们已知

[12xˉ0(x+xˉ02∂∂x)−α]ψα(x)=0\left[\frac{1}{\sqrt{2\bar{x}_0}}\left(x + \bar{x}_0^2 \frac{\partial}{\partial x}\right)-\alpha\right]\psi_{\alpha}(x) = 0 [2xˉ0​​1​(x+xˉ02​∂x∂​)−α]ψα​(x)=0

[(x−2xˉ0α)+xˉ02∂∂x]ψα(x)=0\left[\left(x-\sqrt{2}\bar{x}_0 \alpha\right) + \bar{x}_0^2 \frac{\partial}{\partial x}\right]\psi_{\alpha}(x) = 0 [(x−2​xˉ0​α)+xˉ02​∂x∂​]ψα​(x)=0

相干态的性质

非正交性：相干态不是正交归一的，两个不同的相干态的内积为

⟨α∣β⟩=e−12(∣α∣2+∣β∣2)∑n=0∞α∗nn!βnn!=e−12(∣α∣2+∣β∣2−2α∗β)\langle \alpha|\beta\rangle =e^{-\frac{1}{2}(|\alpha|^2 + |\beta|^2)}\sum_{n=0}^{\infty}\frac{\alpha^{*n}}{\sqrt{n!}}\frac{\beta^{n}}{\sqrt{n!}}=e^{-\frac{1}{2}(|\alpha|^2 + |\beta|^2 - 2\alpha^*\beta)} ⟨α∣β⟩=e−21​(∣α∣2+∣β∣2)n=0∑∞​n!​α∗n​n!​βn​=e−21​(∣α∣2+∣β∣2−2α∗β)

当 ∣α−β∣|\alpha - \beta|∣α−β∣ 很大时，也就是说 α\alphaα 和 β\betaβ 两个点在复平面内距离很远时，⟨α∣β⟩≈0\langle \alpha|\beta\rangle \approx 0⟨α∣β⟩≈0，此时两个相干态近似正交。

过完备性：由于相干态不是正交的，因此不满足一般的完备性关系 ∑n∣n⟩⟨n∣=I\sum_n\ket{n}\bra{n} = \mathrm{I}∑n​∣n⟩⟨n∣=I，但是可以证明，相干态满足过完备关系

∫d2απ∣α⟩⟨α∣=I\int \frac{d^2\alpha}{\pi} |\alpha\rangle \langle \alpha| = \mathrm{I} ∫πd2α​∣α⟩⟨α∣=I

其中 d2α=d(Reα)d(Imα)d^2\alpha = d(\mathrm{Re}\alpha)d(\mathrm{Im}\alpha)d2α=d(Reα)d(Imα)。

相干态的光子数分布（泊松分布）

由于相干态可由一系列 Fock 态叠加而成，因此数算符可直接作用在相干态，同时得到光子数和其概率幅

N^∣α⟩=N^∑n=0∞e−12∣α∣2αnn!∣n⟩=∑n=0∞ne−12∣α∣2αnn!∣n⟩\hat{N}|\alpha\rangle = \hat{N}\sum_{n=0}^{\infty} e^{-\frac12 |\alpha|^2} \frac{\alpha^n}{\sqrt{n!}} |n\rangle = \sum_{n=0}^{\infty} n e^{-\frac12 |\alpha|^2} \frac{\alpha^n}{\sqrt{n!}} |n\rangle N^∣α⟩=N^n=0∑∞​e−21​∣α∣2n!​αn​∣n⟩=n=0∑∞​ne−21​∣α∣2n!​αn​∣n⟩

P(n)=∣⟨n∣α⟩∣2=e−∣α∣2∣α∣2nn!=Nˉnn!e−NˉP(n) = |\langle n|\alpha\rangle|^2 = e^{-|\alpha|^2} \frac{|\alpha|^{2n}}{n!} = \frac{\bar{N}^n}{n!} e^{-\bar{N}} P(n)=∣⟨n∣α⟩∣2=e−∣α∣2n!∣α∣2n​=n!Nˉn​e−Nˉ

这正是泊松分布的形式，且方差 σ2=Nˉ\sigma^2 = \bar{N}σ2=Nˉ。泊松分布是一个自然界中常见的分布，例如我们随机扔小球，单位时间内扔出小球数量是固定的，但是每个时刻接受到的个数会有差异，接收到 nnn 个小球的概率 P(n)P(n)P(n) 就服从泊松分布。

使用统计方法求得泊松分布的平均值为

⟨n2⟩=∑n2Pn=λ2+λ\braket{n^2} = \sum n^2 P_n=\lambda^2+\lambda ⟨n2⟩=∑n2Pn​=λ2+λ

使用统计方法只能计算⟨n⟩\braket{n}⟨n⟩ 和 ⟨n2⟩\braket{n^2}⟨n2⟩这等低阶矩，对于⟨nk⟩\braket{n^k}⟨nk⟩等高阶矩则非常麻烦，这里介绍生成函数的方法简便计算：
定义生成函数为

G(β)=⟨eβn⟩=∑eβnPn=eλ(eβ−1)G(\beta)=\braket{e^{\beta n}}=\sum e^{\beta n}P_n=e^{\lambda(e^{\beta}-1)} G(β)=⟨eβn⟩=∑eβnPn​=eλ(eβ−1)

泊松分布的方差值为

⟨δn2⟩=⟨n2⟩−⟨n⟩2=λ=⟨n⟩\braket{\delta n^2}=\braket{n^2}-\braket{n}^2=\lambda=\braket{n} ⟨δn2⟩=⟨n2⟩−⟨n⟩2=λ=⟨n⟩

方差值等于平均值，又因为泊松分布的宽度等于标准差等于λ\sqrt{\lambda}λ​，那么平均光子数越大，分布越宽，再看标准差与平均数的比值

δN2‾N‾=1N‾\frac{\sqrt{\overline{\delta N^2}}}{\overline{N}}=\frac{1}{\sqrt{\overline{N}}} NδN2​​=N​1​

这表明相对平均光子数而言，宽度是随着平均光子数的增大而减小的。这有什么物理意义呢？我们知道光强就是单位时间内接受到的光子数，而在实验中，激光可以近似于相干态，那么当光强很大时，单位时间内接受到的光子数也很大，此时光子数涨落的绝对值虽然相比于弱光更大，但相对涨落更小。粒子数的这种涨落常被称为散粒噪声，在弱光下散粒噪声尤为明显，散粒噪声代表了任何基于粒子计数测量的最终精度极限，导致了信噪比与 N\sqrt{N}N​ 成正比的普遍规律，只有通过量子态（压缩态）才能突破这个经典极限。
